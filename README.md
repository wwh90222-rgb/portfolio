# 어떻게 실행하나요? (How to Run)

현재 프로젝트 설정은 완료되었으나, **패키지 설치(npm install)**가 되어있지 않아 사이트가 열리지 않습니다.
다음 단계를 따라주세요.

## 1. Node.js 설치 확인
터미널에서 `node -v`와 `npm -v`를 입력해 설치 여부를 확인해주세요.
없다면 [Node.js 공식 홈페이지](https://nodejs.org/)에서 설치해야 합니다.

## 2. 패키지 설치 (필수!)
터미널에서 프로젝트 폴더(`c:\Google_Antigravity`)로 이동한 뒤 아래 명령어를 입력하세요.

```bash
npm install
```

이 과정은 약 1~2분 정도 소요되며 `node_modules` 폴더가 생성됩니다.

## 3. 개발 서버 실행
설치가 끝나면 아래 명령어로 서버를 켭니다.

```bash
npm run dev
```

터미널에 `http://localhost:5173` 같은 주소가 나오면, Ctrl + 클릭하거나 브라우저에 입력해서 접속하세요.

---

### 문제 해결
- **`npm` 명령어를 찾을 수 없나요?** Node.js를 설치하고 터미널을 껐다 켜보세요.
- **빈 화면만 나오나요?** 브라우저 개발자 도구(F12) Console 탭을 확인해보세요.
